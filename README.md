# Digital Clock Project

A modern live digital clock built using HTML, CSS, and JavaScript.
This project displays the current time in real-time with automatic updates every second without refreshing the page.

Features included:

* Live HH : MM : SS clock
* Current day and date.
* AM/PM indicator.
* Dynamic greeting based on time.
* Responsive modern UI.
* Glow and glassmorphism effects.

Concepts Used:

* JavaScript Date Object.
* DOM Manipulation
* setInterval()
* Functions
* Responsive CSS Design

Tech Stack:
HTML | CSS | JavaScript
